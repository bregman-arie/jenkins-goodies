# Jenkins Goodies

A collection of Jenkins resources, scripts and other goodies for you to explore. Enjoy! :)

## Scripts - How to run

There are several ways to run the snippets in this repository. The quickest would be:

    1. Go to https://<Jenkins_URL>/script
    2. Paste the script
    3. Click on "Run"
    4. Enjoy :)
