# Jenkins Goodies

A collection of Jenkins groovy scripts and snippets

## How to run

There are several ways to run the snippets in this repo. The quickest would be:

    1. Go to https://<Jenkins_URL>/script
    2. Paste the script
    3. Click on "Run"
    4. Enjoy :)
